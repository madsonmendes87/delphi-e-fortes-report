# Exemplo de Relatório com Fortes Report (Delphi)

## 📋 Sobre o projeto

Este projeto demonstra como criar e exibir um relatório utilizando o **Fortes Report** em aplicações Delphi.

O exemplo foi desenvolvido para servir como referência na implementação de relatórios, mostrando desde a consulta dos dados até a visualização em modo **Preview**.

## ✨ Funcionalidades

* Consulta de dados através de um `TFDQuery`.
* Exibição dos dados em um relatório do Fortes Report.
* Visualização em Preview.

## 🛠️ Tecnologias utilizadas

* Delphi
* Fortes Report CE
* FireDAC
* PostgreSQL

## 📁 Estrutura do projeto

```text
Projeto
├── Form principal
├── Form do relatório
├── DataModule
```

## 🚀 Como executar

1. Instale o **Fortes Report** em sua versão compatível com o Delphi utilizado.
2. Abra o projeto no Delphi.
3. Configure a conexão com o banco de dados.
4. Compile e execute a aplicação.
5. Clique no botão responsável por abrir o relatório.

## 📷 Resultado

Ao executar o projeto, será exibida a janela de Preview do Fortes Report contendo os dados retornados pela consulta.

## 📌 Objetivo

O foco deste projeto é apresentar uma implementação simples e objetiva do Fortes Report, facilitando o aprendizado e servindo como base para novos projetos.
